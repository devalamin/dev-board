What is the difference between innerHTML, innerText, and textContent ?

There are some notable difference between innerHTML,innerText and textContent.
as we know innerHTML returns all html tags along with all text that is contained by an element.
on the other hand innerText returns all text contained by an element.
and TextContent is all text contained by an element and all its children

What is event delegation in the DOM?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows us to handle events at a higher level in the DOM tree other than the level where the event was first received

What is event bubbling in the DOM?

Event bubbling is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements.if we don't want to trigger the parent elements, we can use stopPropagation()
